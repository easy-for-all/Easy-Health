# PROJECT: Easy Health

## PURPOSE
Webapp para simplificar treino e acompanhamento de saúde, com foco em execução diária, consistência e evolução do usuário.

---
## EXTRA RULES
Antes de implementar, consulte os arquivos em /rules quando forem relevantes.

## TECH STACK
- Frontend: Next.js + React + TypeScript + Tailwind
- Backend: Ruby on Rails API
- Database: PostgreSQL
- Auth: Rails session/cookie (no JWT in MVP)
- Storage: S3-compatible

---

## ARCHITECTURE
- Modular monolith
- API-first
- Business logic lives in backend (Rails)
- Frontend organized by features

---

## MVP SCOPE
Build only:

- login
- onboarding
- workout plan (weekly)
- workout of the day
- exercise swap
- simple activity log

Avoid extra features.
